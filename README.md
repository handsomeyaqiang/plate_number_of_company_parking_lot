# 公司停车场车牌号自动识别系统
此分支项目开发的分支dev
#### 项目介绍
本项目是为了解决现在公司内部员工上下班停车不方便、停车慢、收费慢问题。本项目是一个公司停车场车牌号自动识别系统，能够检测到进出车辆并提取车辆牌照信息（含汉字字符、英文字母、阿拉伯数字及号牌颜色），进行记录，以及收费。
